# ts-challenges

A bunch of general purpose challenges involving TS

## valid_braces

input will be a nonempty set of braces "()[]{}" and the output will be `true` if they're valid or `false` if not

output exemple:
"(){}[]" => True
"([{}])" => True
"(}" => False
"[(])" => False
"[({})](]" => False
